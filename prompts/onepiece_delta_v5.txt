One Piece Trading Card Game from all sets (OP-01 Romance Dawn 2022 through OP-13 Carrying On His Will 2025)

═══════════════════════════════════════════════════════════════════════
A. ONE PIECE CARD INFORMATION EXTRACTION
═══════════════════════════════════════════════════════════════════════

Extract the following fields from One Piece TCG cards:

────────────────────────────────────────────────────────────────────────
REQUIRED FIELDS (All One Piece Cards)
────────────────────────────────────────────────────────────────────────

**player_or_character**: Character name
• One Piece character name from the manga/anime
• Example: "Monkey D. Luffy", "Roronoa Zoro", "Nami", "Trafalgar Law", "Shanks"
• Use full character name as printed on card

**card_name**: Full card title
• Character name as printed, may include version/subtitle
• Example: "Monkey D. Luffy", "Roronoa Zoro", "Shanks"
• Some cards have same character with different versions in different sets

**card_id**: Card set identifier
🚨 **CRITICAL - EXACT OCR REQUIRED**:
• READ THE ACTUAL PRINTED NUMBER ON THE CARD - DO NOT INFER OR CORRECT
• Transcribe EXACTLY what is printed, character-by-character
• Format: "OPXX-XXX" (e.g., "OP01-001", "OP03-122", "ST01-001")
• Located at bottom of card
• DO NOT "fix" numbers based on your knowledge of set sizes
• If partially obscured, report what IS visible with uncertainty note

**card_number**: Collector number (same as card_id for One Piece)
• Same format as card_id: "OP01-001", "ST01-012", etc.

**card_color**: Card color (CRITICAL for One Piece TCG)
• Options: Red, Blue, Green, Purple, Black, Yellow
• Multi-color cards exist (e.g., "Red/Green", "Blue/Purple")
• Determined by card frame/border color

**op_card_type**: Card type
• Options: Leader, Character, Event, Stage
• Leader = deck commander, has Life value, unique gameplay rules
• Character = main card type, has Power and optional Counter
• Event = one-time effect cards
• Stage = permanent field effects

**rarity**: Rarity level
• L = Leader (deck leader cards)
• C = Common
• UC = Uncommon
• R = Rare
• SR = Super Rare (holographic)
• SEC = Secret Rare (premium holographic)
• SP = Special/Promo
• P = Promo

**card_cost**: Cost to play (DON!! cost)
• Numeric value (typically 0-10)
• Located in top-left corner circle
• Leaders have no cost (they start in play)

**card_power**: Power value
• Numeric value (e.g., 5000, 6000, 7000, 8000)
• Located at bottom-left
• Determines combat strength
• Leaders and Characters have Power
• Events and Stages do not have Power

**life**: Life value (Leader cards ONLY)
• Numeric value (typically 3, 4, or 5)
• Only Leader cards have Life
• Located on Leader card
• null for non-Leader cards

**counter_amount**: Counter value (Character cards)
• Options: 1000, 2000, or null (no counter)
• Located at bottom of Character cards
• Used when blocking attacks
• null for Leaders, Events, Stages

**attribute**: Combat attribute
• Options: Slash, Strike, Ranged, Special, Wisdom
• Indicates attack style icon on card
• Some cards have multiple attributes

**sub_types**: Character traits/affiliations
• Example: "Straw Hat Crew", "Supernovas", "The Four Emperors", "Navy"
• Multiple traits possible, comma-separated
• Located in card type line
• Important for deck building synergies

**set_name**: Set/expansion name
• See Set Identification Module below
• Extract from set code prefix

**set_id**: Set code
• Format: "OP-01", "OP-02", "ST-01", etc.
• Derived from card_id prefix

**year**: Release year
• 4-digit year based on set release
• 2022 for early sets, 2023-2026 for recent
• **If the year is not clearly visible from copyright text, a printed date, or an unambiguous set code: return null. Do NOT guess based on the character, art style, or set "feel." A blank year on the label is better than a wrong one.**

**manufacturer**: Card publisher
• "Bandai" for all One Piece TCG cards

**authentic**: Licensed product verification (true|false)
• true if official Bandai One Piece TCG card
• Check for Bandai logo, official card back design
• false if proxy/counterfeit

────────────────────────────────────────────────────────────────────────
VARIANT DETECTION (CRITICAL FOR ONE PIECE)
────────────────────────────────────────────────────────────────────────

**variant_type**: Card variant classification

🚨 **CRITICAL:** One Piece TCG has many valuable variants that share the SAME card number.
A Parallel Shanks (OP01-120) can be worth 100x more than the base version.
Correctly identifying variants is ESSENTIAL for accurate valuation.

VARIANT TYPES (in order of typical value):

• **null/Base**: Standard printing
  - Regular card stock
  - No special treatments
  - Most common version

• **parallel**: Parallel Rare
  - Full-card holographic/foil treatment
  - Rainbow prismatic pattern
  - Distinctly shiny surface
  - Card art same as base but with holo finish

• **manga**: Manga Art variant
  - Black and white manga panel artwork
  - Distinctive manga-style illustration
  - Different art from standard version
  - Usually depicts manga scene

• **parallel_manga**: Parallel + Manga combination
  - Manga art WITH holographic treatment
  - Extremely rare
  - Highest value variants

• **sp**: Special Art
  - Tournament prizes or special releases
  - Unique artwork
  - Often full-art treatment

• **alternate_art**: Alternate Art version
  - Different illustration from base
  - Same card stats/abilities
  - Full-art or extended art

• **box_topper**: Box Topper promo
  - Included with booster box purchases
  - Often oversized or special finish

• **treasure_rare**: Treasure Rare
  - Premium foil variant
  - Special border treatment
  - Very limited

• **super_parallel**: Super Parallel
  - Ultra-rare parallel variant
  - Enhanced holographic effect

• **red_super_alt_art**: Red Super Alternate Art
  - Special red signature series
  - Premium alternate artwork

VARIANT INDICATORS TO LOOK FOR:
- Holographic/rainbow surface = Parallel variant
- Black & white manga panels = Manga variant
- Different artwork composition = Alternate Art
- Special border/frame treatment = SP or Treasure
- "P" in card number = Promo variant

OUTPUT FORMAT:
{
  "variant_type": "parallel",
  "variant_confidence": "high",
  "variant_indicators": ["holographic surface", "rainbow pattern visible"]
}

────────────────────────────────────────────────────────────────────────
ENHANCED FIELDS
────────────────────────────────────────────────────────────────────────

**card_text**: Card abilities and effects
• Extract all card text
• Include [Trigger], [Blocker], [Rush], [DON!!x1] abilities
• Format with line breaks

**rarity_tier**: Visual rarity classification
• Based on card finish and rarity symbol
• Options: Common, Uncommon, Rare, Super Rare, Secret Rare, Special

**subset**: Additional variant/promo designation
• Promo, Tournament, Box Topper, etc.

**is_foil**: Holographic treatment (true|false) — DEFAULT TO FALSE
• true ONLY for cards with clear holographic/prismatic surface treatment
  (Parallel variants, SEC, and some SR cards)
• false for standard printings and when uncertain
• ⚠️ COMMON FALSE POSITIVES: Camera glare, glossy card stock, sleeve reflections,
  and lighting artifacts are NOT foil. Only set true with clear rainbow/prismatic shifting.
• If card is in a protective sleeve/holder: almost always set false

**language**: Card language
• Usually "English" or "Japanese"
• Japanese cards may have different values

C. ONE PIECE CARD DEFECT PATTERNS
═══════════════════════════════════════════════════════════════════════

**Common One Piece TCG Defects:**

────────────────────────────────────────────────────────────────────────
1. PARALLEL/HOLOGRAPHIC DEFECTS (HIGH VALUE CARDS)
────────────────────────────────────────────────────────────────────────

**Parallel Card Characteristics:**
• Full-card prismatic/rainbow holographic treatment
• Entire card surface is holographic
• More susceptible to surface defects
• Highly sought after by collectors

**Common Parallel Issues:**
• Holo scratches across surface (micro-scratches visible under light)
• Pattern disruption or clouding
• Fingerprints visible on holo surface
• Rainbow pattern inconsistency
• Surface scuffing dulling the holo effect

**Grading Impact (v8.8 — score via the master SURFACE SCORING LADDER, Section 10):**
• Hairline holo scratch, angle/zoom only → 9
• Holo scratch or pattern disruption noticeable at normal distance → 7 per the ladder
• Confirmed handling defects prevent a 10 surface score
🚨 GLARE CHECK FIRST: apply the master GLARE & REFLECTION FALSE-POSITIVE CHECK (Section 10,
Step 5) — especially on dark Leader cards and black-bordered cards, whose glossy dark surfaces
reflect light as scuff-like streaks in photos.

────────────────────────────────────────────────────────────────────────
2. COLORED BORDER DEFECTS
────────────────────────────────────────────────────────────────────────

**One Piece Card Borders:**
• Border color matches card color (Red, Blue, Green, etc.)
• Colored borders provide high contrast with white cardstock
• Edge wear shows as white fiber exposure against colored border

**Detection:**
• Examine all four edges for whitening
• Red/Blue/Green borders make whitening highly visible
• Check corners where borders meet

**Grading Impact (v8.8 — score via the master EDGE SCORING LADDER, Section 9; no per-edge stacking):**
• ONE tiny fleck, close-inspection only → 9; 2-3 flecks → 8
• Whitening noticeable at normal distance on part of ONE edge → 7
• Noticeable whitening on multiple edges, or continuous → 6; heavier per the ladder
(The ladder's spread adjustment handles multiple edges — do NOT stack per-edge deductions.)

────────────────────────────────────────────────────────────────────────
3. CORNER DEFECTS
────────────────────────────────────────────────────────────────────────

**One Piece Card Corners:**
• Rounded corners (standard TCG radius)
• Colored border extends to corner edges
• Corner wear common from shuffling/handling

**Common Issues:**
• Corner rounding/blunting from wear
• White fiber exposure at corner tips
• Corner dings or dents
• Peeling at corner layers

**Grading Impact (v8.8 — score via the master CORNER SCORING LADDER, Section 8; no per-corner stacking):**
• ONE corner with pinpoint close-inspection wear → 9; 2+ corners → 8
• Wear noticeable at normal distance (1-2 corners) → 7
• Multiple noticeable corners / obvious blunting → 6; rounding → 5-4 per the ladder
(The ladder's spread adjustment handles multiple corners — do NOT stack per-corner deductions.)

────────────────────────────────────────────────────────────────────────
4. SURFACE DEFECTS
────────────────────────────────────────────────────────────────────────

**Standard Card Surface:**
• Glossy front surface with character artwork
• Standard card back with One Piece logo pattern

**Common Issues:**
• Print lines through artwork
• Scratches across character art (highly visible)
• Surface indentations or pressure marks
• Ink spots or printing errors
• Silvering on dark areas

**Character Art Considerations:**
• Defects on character face/focal point more impactful
• Defects in background less severe
• Center artwork damage most noticeable

**Grading Impact (v8.8 — score via the master SURFACE SCORING LADDER, Section 10):**
• Hairline/close-inspection scratch or print line → 9/8 per the ladder
• Scratch noticeable at normal distance → 7 (location does not change the score — a scratch
  on the character's face is described in prose but scored by visibility like any other)
• Heavy surface damage → 6 and below per the ladder

────────────────────────────────────────────────────────────────────────
5. PRINTING/MANUFACTURING DEFECTS
────────────────────────────────────────────────────────────────────────

**Common Factory Issues:**
• Off-center printing (affects centering grade)
• Ink roller lines
• Color registration errors
• Miscut cards
• Crimped edges from packaging

**Grading Impact (v8.8 — score via the master SURFACE SCORING LADDER, Section 10):**
• Printing variance detectable only on close inspection → 9 (single) / 8 (multiple)
• Print defects noticeable at normal distance → 7 per the ladder
• Miscuts: score centering from the measured ratio via the CENTERING SCORE GUIDE

D. ONE PIECE CARD GRADING EXAMPLES
═══════════════════════════════════════════════════════════════════════

**CENTERING EXAMPLE (Standard Card):**
"This One Piece TCG card features a Red border frame. Measuring borders: left ≈ 2.8mm, right ≈ 3.2mm, top ≈ 2.5mm, bottom ≈ 3.0mm. Left-right ratio: 53/47. Top-bottom ratio: 55/45. Worst axis: top-bottom at 55/45 → centering 10 per the CENTERING SCORE GUIDE."

**PARALLEL VARIANT EXAMPLE:**
"This Shanks Parallel variant (OP01-120) displays full-card prismatic holographic treatment with rainbow pattern visible across the entire surface. Examining the holo layer under varied lighting angles, I detect one fine hairline scratch approximately 8mm in length running diagonally across the upper left quadrant. The scratch causes minor pattern disruption but does not affect the character artwork. Per the visibility test this is a Minor defect (surface ladder row 9). Surface score: 9."

**MANGA VARIANT EXAMPLE:**
"This Monkey D. Luffy Manga variant (OP01-003) features distinctive black and white manga panel artwork depicting a scene from the manga. The card has standard (non-holographic) finish.

🚨 Manga-art note: the deliberately rough, sketchy black-and-white linework and halftone dot
shading are the PRINTED DESIGN — never report them as print lines, scratches, or silvering.

Surface is clean with no visible defects. Edges show minimal handling with a trace of
whitening at the top edge, detectable only on close inspection — Minor severity. As the face's only flaw this is edge ladder row 9. Edge score: 9."

**CORNER ANALYSIS EXAMPLE:**
"Examining all four corners of this Character card at maximum zoom:
- Top-left: Sharp corner point with intact border, no fiber exposure. Pristine.
- Top-right: Minimal softening at the corner tip, detectable only on close inspection. Near pristine.
- Bottom-left: Trace white fiber visible against the Blue border on close inspection. Minor wear.
- Bottom-right: Clean corner, sharp point maintained. Pristine.
Corner score: 8 (Minor close-inspection wear at TWO corners — count-aware corner ladder row 8; a single worn corner would be 9)"

**EDGE ANALYSIS EXAMPLE:**
"Examining all four edges of this Super Rare card:
- Top edge: Clean edge line, Green border intact, no whitening visible. Pristine.
- Bottom edge: A tiny whitening speck near the right corner, close-inspection only; otherwise clean.
- Left edge: No defects, border color consistent throughout.
- Right edge: Two small specks of fiber exposure, detectable only on close inspection.
Edge score: 8 (close-inspection specks on two edges — count-aware edge ladder row 8; a single speck would be 9)"

E. ONE PIECE SPECIFIC EXTRACTION RULES
═══════════════════════════════════════════════════════════════════════

**DON!! CARD DETECTION:**
• DON!! cards are energy cards used in gameplay
• Not typically graded individually
• If detected, note as "DON!! Card" in card_type

**LEADER CARD IDENTIFICATION:**
• Leaders have "Leader" type line
• Leaders have Life value (unique to Leaders)
• Leaders have no cost (start in play)
• Leaders define deck color restrictions

**BLOCKER ABILITY:**
• [Blocker] keyword indicates card can block attacks
• Important for gameplay, note in abilities

**TRIGGER EFFECTS:**
• [Trigger] indicates effect when card is added to life
• Note trigger text in card_text field

**MULTI-COLOR CARDS:**
• Some cards have two colors (e.g., Red/Green)
• Both colors should be listed in card_color
• Format: "Red/Green" or "Blue/Purple"
