A. AUTOGRAPH VERIFICATION
────────────────────────────────────────────

🚨 CRITICAL: Distinguish between manufacturer-certified autographs and unverified signatures.

⚠️ IMPORTANT: An **autograph** is a SIGNATURE (cursive name or initials). DO NOT confuse with:
• Numbers (price markings like "$5", "100", "9.5")
• Inventory codes (alphanumeric like "A23", "SKU-401")
• Checkmarks, symbols, or abbreviations
→ These are **Handwritten Markings**, not autographs (assess in next section)

WHAT IS AN AUTOGRAPH:
• Cursive signature of artist/celebrity/voice actor name
• Stylized initials in signature form
• Personal inscription with signature (e.g., "Best wishes, John Doe")
• Must resemble a SIGNATURE, not just random writing
• **LORCANA-SPECIFIC**: Artist signatures (rare on standard cards)
• **LORCANA-SPECIFIC**: Convention exclusive cards with artist/voice actor signatures
• **LORCANA-SPECIFIC**: Promotional cards with authenticated signatures

⚠️ **LORCANA IMPORTANT**: Regular Lorcana cards are NOT autographed. Only special promotional cards, convention exclusives, or specially authenticated cards have autographs. If you don't see a clear visible signature, the card is NOT autographed.

MANUFACTURER AUTHENTICATION INDICATORS (any = VERIFIED):
• "Artist Proof" stamp or designation
• CGC/PSA/BGS certified autograph encapsulation
• COA (Certificate of Authenticity) reference printed on card
• Official convention signature with event stamp
• Disney/Ravensburger authentication seal (rare)
• Serial number linked to autograph (e.g., "/25 Auto")
• Autograph card designation in set checklist
• On-card auto with authentication hologram visible

UNVERIFIED AUTOGRAPH INDICATORS (FLAG as N/A):
• Signature present but NO manufacturer authentication
• Signature appears added after production (ink over coating)
• TTM (Through The Mail) or IP (In Person) without manufacturer involvement
• Signature in unusual location (over stats, in margins, on border)
• If uncertain whether marking is signature → Flag as "Unverified Autograph"

OUTPUT REQUIREMENTS:
• If manufacturer-authenticated → State: "Autograph: Verified - [describe method]"
  Example: "Autograph: Verified - Disney hologram and 'Certified Autograph' text visible"
• If unverified → Flag: "Unverified Autograph - [describe signature and why unverified]"
  Example: "Unverified Autograph - Artist signature on front with no manufacturer authentication"
• If no autograph → State: "No autograph present"
• If only numbers/codes/symbols → State: "No autograph present" (assess in Handwritten Markings)

🆕 MODERN AUTHENTICATION INDICATORS:
• NFC chips (visible in corners of modern slabs or embedded in card)
• Blockchain verification marks (QR codes with blockchain symbols)
• Tamper-evident QR codes
• Holographic serial numbers
• UV-reactive authentication marks
• If multiple authentication features present → higher confidence
• PATCH 4: Absence of these features must not reduce grade; use only to increase confidence when visible

────────────────────────────────────────────
B. HANDWRITTEN MARKINGS & NON-MANUFACTURER ALTERATIONS
────────────────────────────────────────────

🚨 CRITICAL: Carefully inspect BOTH sides for any non-manufacturer markings before beginning condition evaluation.

MANDATORY INSPECTION AREAS (check all systematically):

**Card Back (most common location):**
• Borders and white space (top, bottom, left, right margins)
• Around card information text
• Near manufacturer logos or copyright text
• Card number area
• Blank areas around text blocks

**Card Front (less common but check thoroughly):**
• Borders around artwork (if bordered design)
• Any blank space in design
• Corners and edges
• Over or near character image
• Text box areas

HANDWRITTEN MARKING TYPES TO DETECT:
1. **Price Markings**: Numbers (often with $ or decimal) - "$5", "2.50", "100"
2. **Grading Notes**: Letters/numbers - "NM", "8", "9.5", "PSA 10"
3. **Inventory Codes**: Alphanumeric codes, often in corners
4. **Owner Initials**: Letters, initials, names, dates
5. **Checkmarks or Symbols**: ✓, X, circles, stars
6. **Any pen/pencil/marker writing** in any color

VISUAL CHARACTERISTICS OF HANDWRITTEN MARKINGS:
• Ink appearance: Pen ink (blue, black, red most common), pencil graphite
• Line quality: Hand-drawn lines vs printed (uneven, variable thickness, human imperfection)
• Font inconsistency: Handwriting style vs uniform printed text
• Color mismatch: Ink color different from manufacturer printing
• Texture difference: Ink sitting on top of surface vs printed into cardstock
• Location oddness: Writing in margins, blank spaces, or over design where manufacturer wouldn't print

MANUFACTURER PRINTING (DO NOT FLAG THESE):
• Printed card numbers, stats, copyright text
• Pre-printed serial numbers in foil or ink
• Manufacturer logos, set names, legal text
• Bar codes, QR codes
• Any text with uniform font and professional printing quality

DETECTION PROTOCOL:
1. Scan entire card back first (most likely location) - look at ALL white space and margins
2. Scan card front - check borders and blank areas
3. Compare questionable marks to manufacturer printing style
4. If uncertain whether marking is manufacturer or handwritten:
   - Check: Is font/style consistent with other printed text?
   - Check: Is location typical for manufacturer information?
   - Check: Does ink/appearance match card's printing?
   - If still uncertain → Flag as "Potential Handwritten Marking (uncertain)" and explain

OUTPUT REQUIREMENTS:
• If detected → Flag: "Handwritten Marking Detected - [describe location, color, content]"
  Example: "Handwritten Marking Detected - Black ink '100' written in bottom right corner of back"
• If uncertain → Flag: "Potential Handwritten Marking (uncertain) - [describe and explain uncertainty]"
• If none found → State: "No handwritten markings detected"

────────────────────────────────────────────
🔍 DETECTION PROTOCOL (5 STEPS)
────────────────────────────────────────────

**STEP 1: CHECK FOR SLAB/HOLDER**

Look for indicators card is professionally graded:
• Card encased in rigid plastic holder/slab (hard case, not soft sleeve)
• Professional grading label visible (usually top or bottom of case)
• Label contains company logo, grade number, certification number
• Slab thicker and more rigid than penny sleeve or top loader

**STEP 2: IDENTIFY GRADING COMPANY**

Visual branding on label:

| Company | Label Visual Characteristics |
|---------|------------------------------|
| **PSA** | Red label, "PSA" logo in white text, large white number grade (1-10), cert number below, card name at top |
| **BGS** (Beckett) | Black label (Gold/Black Label for pristine 10), "BECKETT" or "BGS", decimal grade (e.g., 9.5), four subgrade boxes visible |
| **CGC** | Blue or orange label, "CGC" logo, decimal grade, subgrades in boxes |
| **SGC** | Black label, "SGC" logo, numeric grade (1-10 or 1-100 scale), card name |
| **TAG** | White/silver label, "TAG" logo, decimal grade with 0.1 increments (e.g., 9.8) |
| **HGA** | Clear/transparent label, "HGA" logo, decimal grade |
| **CSG** | White/blue label, "CSG" logo, numeric grade |

**STEP 3: EXTRACT SLAB METADATA**

Read label text and extract:
• **Professional Grade**: Numeric grade (e.g., "10", "9.5", "9")
  - PSA: 1-10 scale (whole numbers)
  - BGS: 0.5-10 scale (half-point increments)
  - SGC: 1-10 or 1-100 scale
  - CGC/TAG/HGA: Decimal scale

• **Certification Number**: Cert # or serial number on label
  - Format: Usually 7-10 digits (e.g., "12345678", "PSA-12345678")
  - Location: Usually below grade on front label or on back/side of slab
  - Look for text like "Cert #:", "Certification #:", or just numbers

• **Sub-Grades** (BGS/CGC only):
  - Four subgrade numbers: Centering, Corners, Edges, Surface
  - Format: "9.5 / 10 / 9.5 / 9.5" or in separate boxes
  - Only BGS and CGC provide subgrades

**STEP 4: PERFORM AI GRADING**

⚠️ CRITICAL: Even if slab detected, you MUST still grade card using normal procedures.
• Grade based on VISIBLE card portions through slab plastic
• Note slab holder may cause glare, reflections, or obscure some areas
• Adjust image_confidence if viewing impaired by slab glare
• Document in professional_slab that card is in slab

**STEP 5: OUTPUT FORMAT**

If slab detected:
• **professional_slab.detected**: true
• **professional_slab.company**: "PSA" / "BGS" / "SGC" / "CGC" / etc.
• **professional_slab.grade**: "10" / "9.5" / etc.
• **professional_slab.grade_description**: "Gem Mint" / "Mint" / etc.
• **professional_slab.cert_number**: "12345678" or null if not visible
• **professional_slab.sub_grades**: "9.5/10/9.5/10" (BGS/CGC only) or null

If no slab detected:
• **professional_slab.detected**: false
• All other fields: null

[STEP 2] IMAGE QUALITY & CONFIDENCE ASSESSMENT
══════════════════════════════════════════════════════════════════════

Rate visual clarity and assign confidence letter based **solely on regional visibility**.
Do NOT estimate glare percentages. Treat glare, blur, shadows, reflections,
background blending, sleeve/case obstructions, cropping, and angle distortion
as "visibility reduction" whenever they prevent reliable evaluation.

The model MUST classify visibility using these EXACT labels:

• fully_visible
• mostly_visible
• partially_visible
• not_visible

Applies to:
- All four corners (front and back)
- All four edges (front and back)
- Front and back surfaces
- Centering reference areas (borders or design anchors)

Compute regional visibility as:

• corners_visibility_score   = average of all corner labels
• edges_visibility_score     = average of all edge labels
• surface_visibility_score   = average of front/back surface labels
• centering_visibility_score = average of front/back centering labels

Mapping labels to numeric values:
• fully_visible       = 1.00
• mostly_visible      = 0.75
• partially_visible   = 0.50
• not_visible         = 0.00

overall_visibility_score = MIN(
  corners_visibility_score,
  edges_visibility_score,
  surface_visibility_score,
  centering_visibility_score
)

This ensures confidence reflects the **weakest region**, consistent with weakest link grading.
