**DELTA PURPOSE:** This file contains ONLY sports card-specific rules.
The master grading rubric (Sections 1-13) applies to ALL sports cards.

**CARD TYPES COVERED:**
Baseball, Basketball, Football, Hockey, Soccer, Wrestling, Boxing, MMA, Racing, Golf, Tennis, and other sports trading cards

═══════════════════════════════════════════════════════════════════════
A. SPORTS CARD INFORMATION EXTRACTION
═══════════════════════════════════════════════════════════════════════

Extract the following fields from sports cards:

────────────────────────────────────────────────────────────────────────
REQUIRED FIELDS (All Sports Cards)
────────────────────────────────────────────────────────────────────────

**card_name**: Full card name/title as printed
• Often includes player name + card type/designation
• Example: "Ken Griffey Jr. Rookie Card"
• Example: "Tom Brady Gold Standard"
• Example: "LeBron James Dominators"

**player_or_character**: Player/athlete name
• Full name as printed on card
• Example: "Ken Griffey Jr."
• Example: "Tom Brady"
• Example: "LeBron James"
• If multiple players (dual/triple card): List all, separated by " / "
• Example: "Michael Jordan / Scottie Pippen"

**set_name**: Set or series name
• Official set name from manufacturer
• Example: "Topps Chrome"
• Example: "Panini Prizm"
• Example: "Upper Deck SP Authentic"
• Example: "Bowman Chrome"

**year**: Year from copyright or release date
• 4-digit year format
• Example: "2024", "2023", "1989"

**YEAR DETECTION HIERARCHY (authoritative → fallback)**:
  1. **Back copyright (© YYYY)** — this is the AUTHORITATIVE print year. If you can read it on the card back, USE IT. Do not override it with anything else.
  2. **Explicit print/release date printed on the card** (front or back).
  3. **Set logo with a 4-digit year** (e.g., "2024 Topps Chrome").
  4. **Front season indicator** (e.g., "2023-24" or "2024-25") — this denotes the SEASON the player played, NOT necessarily the print year. Sports products for "2024-25" are typically printed in 2024 or 2025; the season label can lag the print year by ~1 year. Use the season convention (first year of the range, e.g., "2023" for a "2023-24" card) ONLY if no copyright/print date is visible.

• When the back copyright and the front season label disagree, ALWAYS use the back copyright.
• Modern sports products (Panini, Topps, Upper Deck, Bowman) consistently print a back © year — look for it before falling back to the season label.
• **If the year is not clearly visible from copyright text, a printed date, or an unambiguous set logo: return null. Do NOT guess based on the player, design era, or set "feel." A blank year on the label is better than a wrong one.**

**manufacturer**: Card publisher
• Official company name
• Common manufacturers:
  - Topps (baseball, various)
  - Panini (basketball, football, soccer)
  - Upper Deck (hockey, various)
  - Bowman (baseball)
  - Fleer (vintage)
  - Donruss (vintage, basketball)
  - Score (vintage, various)

**card_number**: Card number within set
🚨 **CRITICAL - NUMBER ONLY, NOT SUBSET**:
• Extract ONLY the numeric card number - DO NOT include subset/parallel names
• If card shows "123 Silver Prizm" → card_number: "123", subset: "Silver Prizm"
• If card shows "#45 Refractor" → card_number: "45", subset: "Refractor"
• If card shows "RC-45 Gold" → card_number: "RC-45", subset: "Gold"
• SEPARATE the number from any variant/parallel/subset text
• Common patterns to EXCLUDE from card_number:
  - Color names: Silver, Gold, Red, Blue, Green, Purple, Orange, Pink, Black
  - Parallel names: Prizm, Refractor, Holo, Chrome, Mosaic, Optic
  - Pattern names: Wave, Shimmer, Disco, Tiger Stripe, Cracked Ice
• Prefixes like "RC-", "AU-", "RPA-" ARE part of the card number
• Examples:
  - "150" → card_number: "150"
  - "#RC-25" → card_number: "RC-25"
  - "AU-TB" → card_number: "AU-TB"
  - "45/299 Silver Prizm" → card_number: "45/299", subset: "Silver Prizm"

**sport_or_category**: Sport category
• Primary sport type
• Options: Baseball, Basketball, Football, Hockey, Soccer, Wrestling, Boxing, MMA, Racing, Golf, Tennis, Track & Field, Other

────────────────────────────────────────────────────────────────────────
ENHANCED FIELDS (Modern Sports Cards 2000+)
────────────────────────────────────────────────────────────────────────

**subset**: Subset/insert/parallel name
• CRITICAL: Modern cards (2010+) frequently have subset/insert/parallel variants
• You MUST actively search for subset indicators in small print areas on EVERY card
• DO NOT default to null without checking borders, card number area, and back panel

**Where to Check:**
1. Back of card (bottom/top/sides) - small text
2. Front borders (left/right edges) - vertical text
3. Near card number area - parallel designation
4. Product info area (back bottom) - insert name

**Common Subset Patterns:**
• Panini Prizm: "Silver Prizm", "Gold Prizm", "Ruby Wave", "Tiger Stripe", "Disco"
• Topps Chrome: "Refractor", "X-Fractor", "Wave Refractor", "Atomic Refractor"
• Panini Mosaic: "Silver Mosaic", "Gold Mosaic", "Stained Glass", "Genesis"
• Panini Optic: "Holo", "Purple Shock", "Orange", "Red", "Blue"
• Inserts: "Round Numbers", "Gold Standard", "Dominators", "My House", "Fireworks"

**Detection Tips:**
• Small font, often includes "Insert"/"Parallel"/"Variant"/"Limited"
• Look for color names (Silver, Gold, Red, etc.)
• Pattern names (Wave, Mosaic, Prizm, etc.)
• Don't confuse with large decorative text or player/team names

**Examples:**
✅ GOOD: Panini Prizm Silver card → Back has "Silver Prizm Parallel" → subset: "Silver Prizm"
✅ GOOD: Topps Chrome Refractor → Side has "Refractor" → subset: "Refractor"
❌ BAD: Base card with no subset text after checking all locations → subset: null

**serial_number**: Serial number if present
• Format: "/99" or "12/99" or "#12/99"
• Location: Usually bottom front (foil stamped) or back lower section
• Hand-numbered: If pen-written, verify manufacturer authentication (not TTM/IP)
• Color-coded foil meanings:
  - Gold foil: Often /10
  - Silver foil: Often /25
  - Black foil: Often /1 (1-of-1)
  - Red foil: Often /5
• Examples: "/99", "45/99", "1/1", "/199", "No. 012 of 099"

**rookie_or_first**: Rookie card designation (true|false)
• true if ANY of these present:
  - "RC" logo/designation printed on card
  - "Rookie Card" text on front or back
  - First-year player card (verify year matches player's rookie season)
  - Official rookie designation from manufacturer
• false if:
  - Veteran player card
  - No rookie indicators
  - Second-year or later card

**rarity_or_variant**: Type/variant classification
• Options: Base, SP (Short Print), SSP (Super Short Print), Parallel, Insert, Autograph, Relic, Promo
• Base: Standard mass-produced card, no special features
• SP: Short Print (serial /26-/99 or known SP from set checklist)
• SSP: Super Short Print (serial /2-/25)
• Parallel: Color/pattern variant (Prizm Silver, Refractor, etc.)
• Insert: Named insert set (Gold Standard, Dominators, etc.)
• Autograph: On-card or sticker autograph
• Relic: Memorabilia card (jersey, bat, patch, etc.)

**authentic**: Licensed product verification (true|false)
• true if ALL present:
  - Official league logos (MLB, NBA, NFL, NHL, MLBPA, NFLPA, NBPA, NHLPA)
  - Licensed manufacturer (Topps, Panini, Upper Deck, Bowman)
  - Professional print quality
  - Correct card stock and finish
• false if:
  - Missing league logos
  - Unknown manufacturer
  - Poor print quality (bootleg indicators)
  - Incorrect fonts or logos

**facsimile_autograph**: Does card have official facsimile autograph? (true|false)
• 🚨 MANDATORY: Read ALL text on card back (including fine print, margins, footers) before determining
• true if card back states "Facsimile Autograph", "Facsimile Signature", "reproduction", "reprint", or similar ANYWHERE
• This is a PRINTED/REPRODUCED signature, NOT hand-signed
• Common in vintage reprints and commemorative sets
• Card is still authentic and official - grade normally
• false ONLY if no facsimile/reproduction disclosure present after reading ENTIRE card back
• IMPORTANT: facsimile_autograph = true means autographed = false (not hand-signed)
• ⚠️ Do NOT trust "Certified Autograph Issue" on front alone - reprints copy this text. ALWAYS verify on back.

**official_reprint**: Is card an official reprint? (true|false)
• 🚨 MANDATORY: Read ALL text on card back (including fine print, margins, footers) before determining
• true if card back states "reproduction", "reprint", "reproduced" ANYWHERE on back
• Common text: "The front of this card is a reproduction...", "Reprint", "Official Reproduction"
• Part of commemorative or archive series (e.g., Topps Archives, All-Time Greats)
• Card is legitimate - grade normally with special feature notation
• false ONLY if no reprint disclosure present after reading ENTIRE card back

────────────────────────────────────────────────────────────────────────
OPTIONAL FIELDS (If Visible)
────────────────────────────────────────────────────────────────────────

**team**: Player's team
• Team name as shown on card
• Example: "Seattle Mariners", "New England Patriots", "Los Angeles Lakers"

**position**: Player position
• Abbreviation or full position
• Example: "QB" (Quarterback), "OF" (Outfielder), "PG" (Point Guard)

**card_back_text**: Descriptive paragraph from card back (2-4 sentences)
• NOT just stats - extract bio/highlights/context text
• Example: "Brady led the Patriots to six Super Bowl victories during his 20-year tenure in New England. His combination of accuracy, leadership, and clutch performance made him one of the greatest quarterbacks of all time."
• null if no descriptive text (stats-only back)

═══════════════════════════════════════════════════════════════════════
B. SPORTS CARD RARITY CLASSIFICATION
═══════════════════════════════════════════════════════════════════════

Assign highest visible tier only (in priority order):

────────────────────────────────────────────────────────────────────────
RARITY TIER HIERARCHY (Check in Order, Assign First Match)
────────────────────────────────────────────────────────────────────────

**1. 1-of-1 / Superfractor**
• Serial number = 1/1
• Unique card, only one in existence
• Often labeled "Superfractor", "1/1", "Printing Plate"
• Extremely rare, highest tier

**2. Super Short Print (SSP)**
• Serial numbered /2 to /25
• Very limited production
• High value, highly collectible

**3. Short Print (SP)**
• Serial numbered /26 to /99
• Limited production run
• More available than SSP but still scarce

**4. Autographed**
• On-card or sticker autograph
• REQUIRES manufacturer authentication (hologram, certification, COA)
• If autograph.present = true AND autograph.type = "manufacturer_authenticated" → rarity_tier = "autographed"
• Unverified autographs = N/A grade (not counted as autograph tier)

**5. Memorabilia / Relic**
• Embedded patch, jersey swatch, bat piece, ball piece, ticket stub, etc.
• Material is manufacturer-authenticated as game-used or event-used
• Look for authenticity statements on card back
• Examples: "Game-Used Jersey", "Event-Worn Patch", "Stadium Seat Relic"

**6. Modern Parallels (Prizm, Mosaic, Refractor, Chrome)**
• Color/pattern variants with holographic or special finishes
• Panini Prizm: Silver, Gold, Ruby Wave, Tiger Stripe, Disco, Neon Green, etc.
• Topps Chrome/Refractor: X-Fractor, Wave, Atomic, Pulsar, Speckle, etc.
• Panini Mosaic: Silver, Gold, Stained Glass, Genesis, Reactive Blue, etc.
• Panini Optic: Holo, Purple Shock, Orange, Red, Blue, Pink, etc.
• Panini Select: Silver, Gold, Copper, Tie-Dye, Zebra, etc.
• Special patterns: Cracked Ice, Nebula, Galaxy, Shimmer, etc.

**7. Parallel / Insert Variant**
• Named insert set or variant
• Examples: "Gold Standard", "Dominators", "My House", "Fireworks", "Round Numbers"
• Different from base design, special subset within product
• May have unique card numbers or design elements

**8. Rookie / Debut / 1st Edition**
• Rookie Card (RC) designation
• Player's first licensed card
• 1st Edition marking (vintage cards)
• Debut card from player's rookie season

**9. Limited Edition**
• Serial numbered /100 to /999
• Limited but not as scarce as SP/SSP
• Still collectible, numbered for authentication

**10. Commemorative / Promo**
• Topps NOW (event-specific, limited time print)
• Panini Instant (on-demand event cards)
• Stadium giveaway promos
• Special event issues (Opening Day, All-Star, World Series, etc.)

**11. Base / Common**
• Mass production, no special features
• Standard set card, widely available
• No autograph, serial, rookie, memorabilia, or named subset

────────────────────────────────────────────────────────────────────────
🚨 CRITICAL TIER SELECTION RULE
────────────────────────────────────────────────────────────────────────

**Autograph Priority:**
• Check for autograph FIRST
• If autograph.present = true AND autograph.type = "manufacturer_authenticated" → rarity_tier MUST be "autographed"
• Then check for other features in priority order

**Subset/Insert Detection:**
• CRITICAL: If card has NAMED SUBSET or INSERT designation → This is an INSERT, NOT base_common
• Set rarity_tier = "parallel_variant" or higher (depending on serial number)
• ONLY use "base_common" if NO special features present:
  - No autograph
  - No serial number
  - No rookie designation
  - No memorabilia
  - No named subset/insert/parallel

**Examples:**
• Card with "Silver Prizm" parallel → rarity_tier = "modern_parallel"
• Card with "/50" serial + autograph → rarity_tier = "autographed" (autograph takes priority)
• Card with "Round Numbers" insert designation → rarity_tier = "parallel_variant"
• Plain base card, no features → rarity_tier = "base_common"

C. SPORTS CARD DEFECT PATTERNS
═══════════════════════════════════════════════════════════════════════

**Common Sports Card Defects - How They Present on Sports Cards:**

────────────────────────────────────────────────────────────────────────
1. CHIPPING ON COLORED BORDERS
────────────────────────────────────────────────────────────────────────

**Why Sports Cards Show This:**
• Sports cards often have dark or colored borders (blue, red, green, black, orange)
• Border ink is surface coating over white cardstock
• When coating chips or wears, white cardstock shows through
• Creates high contrast (dark color + white = very visible)

**Detection:**
• Look for white fiber exposure along edges
• Most common on corners where handling occurs
• Check all four edges for white flecks or continuous whitening
• Dark borders make detection easier (white shows clearly)

**Severity (per the master VISIBILITY TEST, Section 1.9):**
• Minor: 1-3 tiny white flecks detectable only on close inspection
• Moderate: Whitening plainly noticeable at normal viewing distance along part of an edge
• Heavy: Extensive chipping, multiple edges affected, large areas

**Grading Impact (score from the master EDGE SCORING LADDER — Section 9; no separate deltas):**
• 1-3 tiny flecks, close-inspection only → 9
• Noticeable whitening along part of ONE edge, or a single tiny chip → 8
• Noticeable whitening on multiple edges, or continuous whitening → 7
• Heavy chipping → 6 or below per the ladder

────────────────────────────────────────────────────────────────────────
2. TWO-SIDED DESIGN CONSIDERATIONS
────────────────────────────────────────────────────────────────────────

**Sports Card Back Characteristics:**
• Text-heavy design (player stats, bio, career highlights)
• Often includes season stats table
• May have manufacturer info, copyright, legal text
• Centering often measured using text blocks as anchors (not always bordered)
• **IMPORTANT:** Many sports cards have ARTWORK on the back - this is NOT an alteration

═══════════════════════════════════════════════════════════════════════
🚨 SPORTS CARD BACK ARTWORK - NOT ALTERATIONS
═══════════════════════════════════════════════════════════════════════

Many sports cards feature legitimate manufacturer artwork on card backs that may appear hand-drawn or illustrated. This is PRINTED artwork, NOT post-production alterations.

**Common Sports Card Back Artwork (Grade Normally - NOT Alterations):**
• Player caricatures or cartoon illustrations
• Team mascots and artistic logo renderings
• Decorative borders with hand-drawn style elements
• Commemorative tributes with artistic flourishes
• Facsimile signatures as part of printed design
• Vintage-style artwork (especially retro/throwback sets)
• Insert set themed graphics and illustrations
• Background artwork, patterns, or artistic textures
• Player portraits in illustrated/painted style
• Action scene illustrations or comic-style artwork

**Sports Card Sets Known for Artistic Backs:**
• Topps Gallery (painted player portraits)
• Donruss Diamond Kings (illustrated player art)
• Topps Gypsy Queen (vintage-style artwork)
• Topps Allen & Ginter (illustrated mini cards)
• Upper Deck Masterpieces (museum-style art)
• Panini Court Kings (basketball artistic series)
• 1980s-1990s Topps/Donruss/Fleer (cartoon mascots, player illustrations)
• Topps Heritage (retro designs with period-appropriate artwork)
• Panini Prizm/Select inserts (themed artistic graphics)

**How to Verify It's Manufacturer Artwork:**
1. Print quality matches rest of card (same halftone dot pattern)
2. Artwork is UNDER the gloss layer, not sitting on top
3. Colors use same palette as overall card design
4. Design appears on other cards in the same set/series
5. Professional execution with clean, precise lines
6. Integrated into card layout (not randomly placed)

**Key Question:** "Is this artwork PRINTED as part of the card, or ADDED by hand after production?"
→ If PRINTED (same print layer, professional quality) = Grade normally
→ If ADDED (on top of gloss, irregular execution) = Flag as alteration

**DO NOT flag as alterations:**
❌ Illustrated player portraits on card backs
❌ Cartoon mascots or team characters
❌ Decorative artistic borders or frames
❌ Facsimile signatures that are clearly printed
❌ Commemorative artwork or tributes
❌ Any artwork that appears on other cards in the set

═══════════════════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════════════════
🚨 SIMULATED-DAMAGE / DISTRESSED DESIGNS - NOT SURFACE DEFECTS
═══════════════════════════════════════════════════════════════════════

Some cards INTENTIONALLY print a worn, aged, or "damaged" aesthetic as part
of the design. These printed effects must NOT be scored as surface defects,
scratches, scribbles, smudges, stains, or alterations.

**Common simulated-damage design elements (Grade Normally - NOT defects):**
• Crumpled, wrinkled, or creased-paper textures (printed shading, not real folds)
• Torn, burnt, or rough/deckled edge artwork
• Ruled-notebook, graph-paper, or legal-pad backgrounds
• Handwritten, scribbled, marker, or "doodle" style text and signatures
• Printed ink smudges, splatter, fingerprints, or stamp/postmark graphics
• Coffee/water-stain, sepia, parchment, or "aged document" effects
• Chalkboard, blueprint, grunge, distressed, or vintage-throwback textures
• UNIFORM tan/beige/cream/parchment panel coloring — this is the design's palette. Do NOT
  describe a uniformly colored text panel as "toning," "print wear," "discoloration," or
  "staining." Real toning/staining is IRREGULAR: uneven patches, blotchy edges, or gradients
  that differ from the rest of the panel. If the tone is even across the whole panel, it is
  the printed design.

🚨 **CRITICAL TEST - printed design vs. physical damage:**
Before flagging ANY mark, line, smudge, scribble, scuff, or "crease" on a
text/graphic panel, ask:
1. Does it sit UNDER the gloss layer (printed) rather than on top of it?
2. Does it share the card's halftone dot pattern and color palette?
3. Does it form a COHERENT, intentional aesthetic across the whole panel?
4. Would the IDENTICAL "damage" appear on every copy of this card?
5. Is the cardstock plane physically UNDISTURBED (no real fold/scratch/indent)?
→ If YES to these = PRINTED DESIGN. No surface deduction. Do not describe it
  as "marks," "smudges," "scribble-like lines," or "alteration."
→ Physical damage instead: scratch BREAKS the gloss, scuff DULLS the finish,
  crease DISTURBS the surface plane, stain has irregular edges and sits on top.

**Sets/issues known for distressed or handwritten-style designs:**
• 1990s "Trade Card"/promo and oddball inserts
• Retro/throwback, grunge, and "vintage reproduction" inserts
• Notebook/sketch/doodle-themed inserts and parallels

🚨 **THREE-PASS NOTE:** Consensus does NOT protect against this. All passes see
the same printed design and will AGREE it "looks marked." Apply the test above
on EVERY pass; do not let a uniformly stylized panel cap the surface subgrade.

**DO NOT flag as defects:**
❌ Printed crumpled/aged-paper texture on the back panel
❌ Handwritten or scribble-style printed text
❌ Printed ink smudges, splatter, or stamp graphics
❌ Any "damage" that is clearly under the gloss and part of the layout

═══════════════════════════════════════════════════════════════════════

**Centering Assessment:**
• Front: Usually has borders or photo framing for measurement
• Back: May be borderless - use text blocks, stats table, or logos as centering anchors
• Text boxes serve as internal design anchors for ratio calculation

**Print Quality Differences:**
• Front: Glossy, photo-quality, high-end finish
• Back: May be matte, text-focused, different coating
• Front surface defects more impactful (visible display side)
• Back surface defects still count — final sub-grade = MIN(front, back) per weakest-link principle

────────────────────────────────────────────────────────────────────────
3. PRINT LINES ON GLOSSY SURFACES
────────────────────────────────────────────────────────────────────────

**High-Gloss Sports Cards:**
• Chrome, Prizm, Optic, Mosaic, Select - all have high-gloss finishes
• Glossy surfaces show print defects more readily than matte
• Print lines are manufacturing artifacts, not handling damage

**⚠️ CRITICAL: DESIGN PATTERN LINES vs. PRINT LINE DEFECTS:**
Many modern sports card parallels feature INTENTIONAL line/geometric patterns as part of their design:
• **Topps Chrome Geometric**: Prominent geometric line pattern across entire card — lines ARE the design
• **Panini Mosaic**: Geometric mosaic/tile pattern — angular lines ARE the design
• **Prizm Wave/Shimmer**: Wavy holographic line patterns — waves ARE the design
• **Select Tie-Dye/Zebra**: Swirling or striped patterns — stripes ARE the design
• **Cracked Ice**: Ice-crack line patterns — fracture lines ARE the design
• **Topps Chrome X-Fractor**: X-shaped refractor pattern — crossing lines ARE the design

🚨 **DO NOT flag design pattern lines as "print lines":**
• If a line is part of a REPEATING geometric/holographic pattern visible across the card → NOT a defect
• If one line in the pattern appears MORE PROMINENT due to lighting angle → NOT a defect (holographic effect)
• If 3/3 or all cards of same parallel type show same "line" in same pattern → It's a design feature, not a defect
• ONLY flag a line as a print line defect if it is ISOLATED, does NOT match the card's pattern design, and appears as an anomaly

**Identifying ACTUAL Print Lines (True Defects):**
• Isolated line that does NOT repeat in a pattern across the card
• Runs contrary to or independent of the holographic/parallel design pattern
• Typically horizontal (from printing rollers), consistent width
• Appears as an anomaly against the card's intended finish
• Present regardless of viewing angle (unlike holographic pattern lines that shift)

**Common Print Line Locations (when they ARE actual defects):**
• Along player image edges (where photo meets background)
• In solid color areas (sky, background, uniform colors)
• Horizontal or vertical roller lines from printing process

**Differentiate from Creases:**
• Print line: Surface-level ink defect, doesn't break cardstock, straight line, consistent width
• Crease: Structural damage, cardstock folded, visible at angles, disrupts surface plane

**Grading Impact (for CONFIRMED actual print line defects only):**
• Single print line: -1 (manufacturing defect, not handling)
• Multiple print lines: -1 to -2
• Print line does NOT trigger structural damage caps (not a crease)
• Design pattern lines that are part of the card's parallel/variant: NO DEDUCTION (not defects)

────────────────────────────────────────────────────────────────────────
4. HOLOFOIL/REFRACTOR PATTERN DEFECTS
────────────────────────────────────────────────────────────────────────

**Sports Cards with Holographic Patterns:**
• Prizm: Rainbow holographic pattern across entire card
• Refractor: Wavelike rainbow effect (Topps Chrome)
• Chrome: Metallic reflective finish with rainbow
• Chrome Geometric: Geometric line pattern overlaying chrome finish — lines are INTENTIONAL DESIGN
• Mosaic: Geometric holographic pattern — mosaic tiles/lines are INTENTIONAL DESIGN
• Optic: Holographic coating with color variants
• X-Fractor: X-shaped crossing refractor pattern — X lines are INTENTIONAL DESIGN

**⚠️ CRITICAL: UNDERSTAND THE CARD'S INTENDED PATTERN BEFORE GRADING:**
Before reporting ANY pattern-related defect on a holographic/parallel card:
1. IDENTIFY the card's parallel type (Geometric, Mosaic, Wave, Prizm, Refractor, etc.)
2. UNDERSTAND that parallel's intended pattern (geometric lines, waves, mosaic tiles, etc.)
3. ONLY flag defects that DEVIATE FROM the intended pattern (scratches through it, missing sections, dead spots)
4. A design line appearing prominent due to lighting/photo angle is the holographic effect working as intended — NOT a defect

**Common Holofoil Defects (ACTUAL defects, not design features):**
• **Holo scratches**: Scratches THROUGH holographic layer that BREAK the pattern (not lines that ARE the pattern)
• **Holo bleed**: Holographic pattern extending outside intended area
• **Pattern disruption**: Missing or damaged sections — dull/dead spots where pattern SHOULD be but ISN'T
• **Foil peeling**: Holographic layer separating from cardstock (rare but serious)

**Detection Method:**
• View card at angle to see holographic layer clearly
• Tilt card under light - holo defects catch light differently
• Scratches will appear as breaks in rainbow pattern
• Pattern disruption visible as dull/non-reflective areas
• Design lines that shift uniformly with viewing angle = holographic design working correctly (NOT defects)

**Grading Impact:**
• Single holo scratch (hairline): -1 minimum
• Multiple holo scratches: -1 to -2
• Pattern disruption (missing holo section): -1 to -2
• Holofoil defects prevent 10 surface score (high-end product expectation)

────────────────────────────────────────────────────────────────────────
5. PATCH/RELIC WINDOW CONSIDERATIONS
────────────────────────────────────────────────────────────────────────

**Memorabilia Cards with Cutout Windows:**
• Window cut into card to embed jersey swatch, bat piece, patch, etc.
• Window edges are factory-cut, not grading defects
• Material visible through window is intentional

**What to Grade:**
• Window edges: Should be clean factory cut, not frayed/damaged
• Area around window: Check for wear, corner damage near cutout
• Surface: Grade normally (avoid window area itself)
• Centering: Measure card centering independently of window placement

**Window Edge Issues (Actual Defects):**
• Fraying around window perimeter (cardstock damage) = defect
• Chipping of ink coating near window = defect
• Window material loose or falling out = serious defect

**NOT Defects:**
• Window itself (intentional design)
• Material texture visible through window (jersey weave, bat grain)
• Window placement off-center (judge overall card centering, not window position)

────────────────────────────────────────────────────────────────────────
6. DIE-CUT SPORTS CARDS
────────────────────────────────────────────────────────────────────────

**Die-Cut Design:**
• Shaped edges (not rectangular)
• Removed corners or decorative cuts
• Jagged/rounded edges by design

**Centering Assessment:**
• Traditional border centering impossible (no straight edges)
• Use internal geometric centering only
• Identify design anchors (player image, text, logos)
• Maximum centering score: 9 (perfect centering difficult to verify)

**Edge Grading:**
• Die-cut edges are intentional, not defects
• Grade edge condition (smoothness of die-cut, no tearing)
• Check for fraying or damage to shaped edges
• Clean die-cut = no deduction; damaged die-cut = defect

────────────────────────────────────────────────────────────────────────
7. VINTAGE SPORTS CARD ISSUES (Pre-2000)
────────────────────────────────────────────────────────────────────────

**Era-Specific Expectations:**
• 1950s-1970s: Rougher factory cuts common, centering often poor
• 1980s: Improved quality but still variable centering
• 1990s: Better cutting, glossy finishes introduced

**Common Vintage Defects:**
• Factory cut roughness (not handling damage) - minimal deduction for era
• Wax staining (cards stored in wax packs) - yellowing or residue
• Print registration issues (colors misaligned) - manufacturing defect
• Centering issues (off-center common, 70/30 or worse typical)

**Grading Adjustments:**
• Don't expect 10 corners on vintage (manufacturing limitations)
• Centering is ALWAYS scored from the master CENTERING SCORE GUIDE table — era does NOT
  change the ratio→score mapping. Era context ("well centered for a 1970s issue") belongs
  in the narrative only.
• Factory roughness noted but less penalized than modern cards
