**DELTA PURPOSE:** This file contains ONLY Yu-Gi-Oh! TCG card-specific rules.
The master grading rubric (Sections 1-13) applies to ALL Yu-Gi-Oh! cards.
This delta provides card-specific extraction rules, defect patterns, and examples.

**CARD TYPES COVERED:**
Yu-Gi-Oh! Trading Card Game from all sets (Legend of Blue Eyes White Dragon 2002 through present 2026)

═══════════════════════════════════════════════════════════════════════
A. YU-GI-OH! CARD INFORMATION EXTRACTION
═══════════════════════════════════════════════════════════════════════

Extract the following fields from Yu-Gi-Oh! TCG cards:

────────────────────────────────────────────────────────────────────────
REQUIRED FIELDS (All Yu-Gi-Oh! Cards)
────────────────────────────────────────────────────────────────────────

**card_name**: Full card name
• Exact name as printed on the card
• Example: "Dark Magician", "Blue-Eyes White Dragon", "Pot of Greed"
• Use full name including punctuation (hyphens, apostrophes, colons)
• Example: "D/D/D Doom King Armageddon", "Number 39: Utopia"

**player_or_character**: Same as card_name for Yu-Gi-Oh!
• Use the card name since Yu-Gi-Oh! cards are the character

**card_id**: Card set code identifier
🚨 **CRITICAL - EXACT OCR REQUIRED**:
• READ THE ACTUAL PRINTED CODE ON THE CARD - DO NOT INFER OR CORRECT
• Transcribe EXACTLY what is printed, character-by-character
• Format: "SET-LANGXXX" (e.g., "LOB-EN005", "DUEA-EN049", "CT13-EN003")
• Located at bottom-right of card (near the passcode number)
• Some older cards use format without language: "LOB-005", "SDK-001"
• DO NOT "fix" codes based on your knowledge
• If partially obscured, report what IS visible with uncertainty note

**card_number**: Same as card_id for Yu-Gi-Oh!
• The set code uniquely identifies a printing

**ygo_card_type**: Card type classification
• **Normal Monster** - Yellow frame, no effect text (italic flavor text)
• **Effect Monster** - Orange frame, has effect text
• **Ritual Monster** - Blue frame, summoned via Ritual Spell
• **Fusion Monster** - Purple frame, Extra Deck card
• **Synchro Monster** - White frame, Extra Deck card
• **XYZ Monster** - Black frame, Extra Deck card with Rank (not Level)
• **Pendulum Monster** - Half monster/half spell with Pendulum Scale
• **Link Monster** - Dark blue frame, has Link Rating and Link Arrows
• **Spell Card** - Green frame (Normal, Continuous, Equip, Quick-Play, Field, Ritual)
• **Trap Card** - Purple/magenta frame (Normal, Continuous, Counter)
• **Token** - Gray frame

**ygo_frame_type**: Frame type for visual identification
• "normal", "effect", "ritual", "fusion", "synchro", "xyz", "pendulum", "link", "spell", "trap", "token"

**ygo_attribute**: Monster attribute (Monster cards ONLY)
• Options: DARK, LIGHT, FIRE, WATER, EARTH, WIND, DIVINE
• Located at top-right of card as an icon
• DIVINE is exclusive to Egyptian God Cards
• Spell/Trap cards have "SPELL" or "TRAP" instead of an attribute

**ygo_race**: Monster type (Yu-Gi-Oh! calls this "Type" on the card)
• Example: "Spellcaster", "Dragon", "Warrior", "Machine", "Fiend", "Fairy"
• Located in the Type line: "[Spellcaster/Effect]"
• For Spell/Trap: "Normal", "Continuous", "Quick-Play", "Equip", "Field", "Counter", "Ritual"

**ygo_atk**: Attack points (Monster cards ONLY)
• Numeric value (e.g., 2500, 3000, 0)
• Located at bottom-left: "ATK/2500"
• Link Monsters have ATK but no DEF
• Spell/Trap cards: null
• Some cards show "?" for variable ATK

**ygo_def**: Defense points (Monster cards ONLY, NOT Link Monsters)
• Numeric value (e.g., 2100, 0)
• Located at bottom-right: "DEF/2100"
• Link Monsters do NOT have DEF
• Spell/Trap cards: null
• Some cards show "?" for variable DEF

**ygo_level**: Level or Rank
• Level: Number of stars (1-12) at top-right for standard monsters
• Rank: Number of stars at top-left for XYZ Monsters (they have Rank, not Level)
• Link Monsters, Spell, and Trap cards: null

**ygo_scale**: Pendulum Scale (Pendulum Monsters ONLY)
• Numeric value (0-13) shown in the blue/red scale diamonds
• Left and right scales are usually the same
• null for non-Pendulum cards

**ygo_linkval**: Link Rating (Link Monsters ONLY)
• Numeric value (1-6) shown in bottom-right where DEF would be: "LINK-3"
• null for non-Link cards

**ygo_archetype**: Archetype/series
• Card series the card belongs to
• Example: "Dark Magician", "Blue-Eyes", "HERO", "Branded", "Spright"
• Not all cards belong to an archetype
• null if no clear archetype

**rarity**: Rarity level
• Common (C) - No foil treatment
• Rare (R) - Silver name text
• Super Rare (SR) - Holographic artwork
• Ultra Rare (UR) - Gold name text + holographic artwork
• Secret Rare (ScR) - Diagonal rainbow holographic pattern on name and art
• Ultimate Rare (UtR) - Embossed/3D textured card surface
• Ghost Rare (GhR) - 3D holographic with "ghost" image
• Starlight Rare (StR) - Prismatic secret pattern across entire card
• Collector's Rare (CR) - Rainbow foil across entire card
• Quarter Century Secret Rare (QCSR) - 25th anniversary special
• Prismatic Secret Rare (PScR) - Enhanced prismatic pattern
• Gold Rare (GUR) - Gold foil treatment
• Premium Gold Rare (PGR) - Enhanced gold foil
• Short Print (SP) - Standard rarity but limited print run

**card_cost**: Not applicable for Yu-Gi-Oh! (set to null)

**set_name**: Set/expansion name
• See Set Identification Module below
• Extract from set code prefix

**year**: Release year
• 4-digit year based on set release
• 2002 for earliest sets, through 2026
• **If the year is not clearly visible from copyright text, a printed date, or an unambiguous set code: return null. Do NOT guess based on the monster, art style, or set "feel." A blank year on the label is better than a wrong one.**

**manufacturer**: Card publisher
• "Konami" for all Yu-Gi-Oh! TCG cards

**authentic**: Licensed product verification (true|false)
• true if official Konami Yu-Gi-Oh! TCG card
• Check for Konami logo, official card back (brown swirl design), holographic eye stamp
• Eye of Anubis hologram present on all authentic cards (bottom-right on newer cards)
• false if proxy/counterfeit

────────────────────────────────────────────────────────────────────────
EDITION DETECTION
────────────────────────────────────────────────────────────────────────

**edition_type**: Card edition
• **1st Edition** - "1st Edition" text below card art on left side
  - Higher value than Unlimited
  - Only on initial print run
• **Limited Edition** - "Limited Edition" text
  - Promo and special releases
• **Unlimited** - No edition text
  - Standard reprint/later printing

EDITION INDICATORS:
- "1st Edition" text on card face = 1st Edition
- "Limited Edition" text = Limited Edition
- No edition marking = Unlimited Edition
- Duel Terminal cards: "DT" in set code

OUTPUT FORMAT:
{
  "edition_type": "1st Edition",
  "edition_confidence": "high"
}

────────────────────────────────────────────────────────────────────────
ENHANCED FIELDS
────────────────────────────────────────────────────────────────────────

**card_text**: Card effect/flavor text
• Effect Monsters: Describe the card's effect in full
• Normal Monsters: Italic flavor text (not a game effect)
• Spell/Trap: Full effect text
• Pendulum: Both Pendulum Effect and Monster Effect

**rarity_tier**: Visual rarity classification
• Based on foil treatment and rarity symbol
• Options: Common, Rare, Super Rare, Ultra Rare, Secret Rare, Ultimate Rare, Ghost Rare, Starlight Rare

**subset**: Additional designation
• Structure Deck, Starter Deck, Tin, Special Edition, etc.

**is_foil**: Holographic treatment (true|false) — DEFAULT TO FALSE
• true ONLY for cards with clear holographic surface treatment
  (Super Rare and above typically have foil/holo)
• false for Common and Rare (silver name only) cards
• ⚠️ COMMON FALSE POSITIVES: Camera glare, glossy card stock, sleeve reflections
• If uncertain: set false

**language**: Card language
• English, Japanese, Korean, German, French, Italian, Spanish, Portuguese
• Japanese cards typically have different set codes and layout

C. YU-GI-OH! CARD DEFECT PATTERNS
═══════════════════════════════════════════════════════════════════════

**Common Yu-Gi-Oh! TCG Defects:**

────────────────────────────────────────────────────────────────────────
1. HOLOGRAPHIC/FOIL DEFECTS (SUPER RARE AND ABOVE)
────────────────────────────────────────────────────────────────────────

**Holographic Card Characteristics:**
• Super Rare: Holographic artwork only
• Ultra Rare: Gold name + holographic artwork
• Secret Rare: Diagonal rainbow pattern on name and art
• Ultimate Rare: Embossed 3D texture across card
• Starlight Rare: Full prismatic treatment

**Common Holo Issues:**
• Holo scratches across artwork (micro-scratches visible under light)
• Fingerprints visible on holo surface
• Genuine flattened/crushed spots interrupting Ultimate Rare embossing
• Scratches that BREAK the Starlight prismatic pattern

🚨 **EMBOSSED/PRISMATIC FINISHES ARE THE DESIGN, NOT DAMAGE (v8.8):**
• Ultimate Rares have an INTENTIONAL 3-D embossed relief on artwork/name/borders — the raised,
  uneven texture is manufactured. Do NOT report the embossing itself as damage; only deduct
  where the relief is visibly CRUSHED or FLATTENED in a localized spot.
• Secret Rares' diagonal rainbow shimmer and Starlight Rares' prismatic sparkle are the
  intended finish. Uniform "clouding" or shimmer across the pattern is the FOIL WORKING —
  only deduct for defined dull dead spots or scratch lines that ignore the pattern.
• Apply the master GLARE & REFLECTION FALSE-POSITIVE CHECK (Section 10, Step 5) before ANY
  holo-surface deduction.

**Grading Impact (v8.8 — score via the master SURFACE SCORING LADDER, Section 10):**
• Hairline holo scratch, angle/zoom only → 9
• Holo scratch or crushed embossing noticeable at normal distance → 8/7 per the ladder
• Confirmed handling defects prevent a 10 surface score

────────────────────────────────────────────────────────────────────────
2. BORDER/EDGE DEFECTS
────────────────────────────────────────────────────────────────────────

**Yu-Gi-Oh! Card Borders:**
• Monster cards have colored frames (yellow, orange, purple, white, black, blue, dark blue)
• Spell cards: green border
• Trap cards: purple/magenta border
• All have outer gold/brown border edge
• White cardstock exposed at edges = edge wear

**Detection:**
• Examine all four edges for whitening
• Gold/brown border makes whitening visible
• Check corners where borders meet

**Grading Impact (v8.8 — score via the master EDGE SCORING LADDER, Section 9; no per-edge stacking):**
• 1-3 tiny flecks, close-inspection only → 9
• Whitening noticeable at normal distance on part of ONE edge → 8
• Noticeable whitening on multiple edges, or continuous → 7; heavier per the ladder
🚨 DARK BACK NOTE: Yu-Gi-Oh's dark brown backs make edge whitening HIGHLY visible — a tiny
speck is still Minor (9) even though it stands out against the dark back. Score by the
visibility test, not by contrast. The glossy dark back is also glare-prone: apply the glare
check before reporting back-surface scuffs.

────────────────────────────────────────────────────────────────────────
3. CORNER DEFECTS
────────────────────────────────────────────────────────────────────────

**Yu-Gi-Oh! Card Corners:**
• Square corners (NOT rounded like Pokemon/One Piece)
• More prone to damage than rounded corners
• Gold border extends to corner points
• Corner wear very common from shuffling

**Common Issues:**
• Corner softening/blunting
• White fiber exposure at corner points
• Corner dings, bends, or dents
• Peeling at corner layers
• "Dog ear" bends at corners

**Grading Impact (v8.8 — score via the master CORNER SCORING LADDER, Section 8; no per-corner stacking):**
• Pinpoint tip whitening/softening, close-inspection only → 9
• Wear noticeable at normal distance → 8; obvious blunting → 7; rounding → 6 and below
• Square corners make wear easier to SEE — the visibility test still decides the band
(The ladder's spread adjustment handles multiple corners — do NOT stack per-corner deductions.)

────────────────────────────────────────────────────────────────────────
4. SURFACE DEFECTS
────────────────────────────────────────────────────────────────────────

**Standard Card Surface:**
• Glossy front with card artwork
• Standard brown swirl card back (consistent across all Yu-Gi-Oh! cards)

**Common Issues:**
• Print lines through artwork
• Scratches across card art
• Surface indentations or pressure marks
• Ink spots or printing errors
• Silvering on dark card areas
• "Hologram" stamp wear (Eye of Anubis on newer cards)

**Card Art Considerations:**
• Defects on monster/character face area more impactful
• Defects on card name text area noticeable
• Background damage less severe but still graded

**Grading Impact (v8.8 — score via the master SURFACE SCORING LADDER, Section 10):**
• Hairline/close-inspection scratch or print line → 9/8 per the ladder
• Scratch noticeable at normal distance → 7 (location — e.g. the monster's face — is described
  in prose but does not change the score)
• Damaged/scratched square hologram stamp (bottom-right) → Minor 9 / Moderate 8 by visibility

────────────────────────────────────────────────────────────────────────
5. PRINTING/MANUFACTURING DEFECTS
────────────────────────────────────────────────────────────────────────

**Common Factory Issues:**
• Off-center printing (affects centering grade)
• Ink roller lines (common in older sets)
• Color registration errors
• Miscut cards (may be collectible if severe)
• Factory crimped edges from packaging

**Grading Impact (v8.8 — score via the master SURFACE SCORING LADDER, Section 10):**
• Printing variance detectable only on close inspection → 9
• Print defects noticeable at normal distance → 8/7 per the ladder
• Miscuts: score centering from the measured ratio via the CENTERING SCORE GUIDE

D. YU-GI-OH! CARD GRADING EXAMPLES
═══════════════════════════════════════════════════════════════════════

**CENTERING EXAMPLE (Standard Card):**
"This Yu-Gi-Oh! card features a yellow Normal Monster frame with gold outer border. Measuring borders: left ≈ 2.5mm, right ≈ 3.0mm, top ≈ 2.2mm, bottom ≈ 2.8mm. Left-right ratio: 45/55. Top-bottom ratio: 44/56. Worst axis: top-bottom at 44/56. Acceptable centering with slight bottom-heavy shift."

**SECRET RARE EXAMPLE:**
"This Blue-Eyes White Dragon Secret Rare (CT13-EN003) displays diagonal rainbow holographic pattern across card name and artwork. Examining the holo layer under varied lighting, I detect two fine hairline scratches across the dragon artwork — visible only when the light catches them, and they cut across the diagonal pattern rather than shifting with it, confirming physical damage. The lower third shows uniform shimmer consistent with the Secret Rare finish (not damage — see the embossed/prismatic design rule). Two hairline scratches, angle-only = Minor severity (surface ladder row 9). Surface score: 9."

**1ST EDITION EXAMPLE:**
"This Dark Magician 1st Edition (LOB-005) from Legend of Blue Eyes White Dragon displays '1st Edition' text clearly below the card artwork. The card shows corner softening with trace whitening at two corner points, noticeable on close inspection, and light whitening along the top edge. Surface is clean with minor handling marks. (v8.8 note: era NEVER adjusts scores — a 2002 card is scored by the same ladders as a 2024 card; age context belongs in prose only.)"

**CORNER ANALYSIS EXAMPLE:**
"Examining all four SQUARE corners of this Effect Monster at maximum zoom:
- Top-left: Light softening at corner point, close-inspection only. Minor (row 9).
- Top-right: Sharp corner maintained, gold border intact. Near pristine.
- Bottom-left: Rounding with white fiber exposure, noticeable at normal distance. Moderate (row 8).
- Bottom-right: Light corner ding, slight depression visible on close inspection. Minor (row 9).
Corner score: 8 (worst corner = bottom-left Moderate row 8; wear on only 2 of 8 corners → no
spread adjustment; whole numbers only — never 8.5)"

**EDGE ANALYSIS EXAMPLE:**
"Examining all four edges of this Ultra Rare card:
- Top edge: Clean edge line, gold border intact. One small whitening speck near center, close-inspection only.
- Bottom edge: A tiny whitening speck near the left corner, close-inspection only.
- Left edge: No defects, border color consistent throughout. Pristine.
- Right edge: A pinpoint chip at one spot, close-inspection only.
Edge score: 8 (worst edge = right edge single tiny chip, row 8; Minor wear spread across 3
edges triggers no further adjustment since the worst state is on only 1 edge)"

E. YU-GI-OH! SPECIFIC EXTRACTION RULES
═══════════════════════════════════════════════════════════════════════

**PASSCODE NUMBER:**
• 8-digit number at bottom-left of card (e.g., 46986414)
• This is the card's unique identifier, NOT the set code
• Do not confuse with the set code (top-right or bottom-right)

**LINK MONSTER IDENTIFICATION:**
• Dark blue/navy frame
• Link Arrows (red triangles pointing in directions around the card)
• "LINK-X" instead of DEF value (e.g., "LINK-3")
• No Level stars — has Link Rating instead
• Record Link Arrows directions: Top, Bottom, Left, Right, Top-Left, Top-Right, Bottom-Left, Bottom-Right

**PENDULUM MONSTER IDENTIFICATION:**
• Split card design: top half is monster, bottom half shows Pendulum Effect
• Two colored diamond scales (left and right) showing Pendulum Scale number
• Can be Normal, Effect, Fusion, Synchro, XYZ, or Ritual Pendulum
• Record both the Pendulum Effect and Monster Effect

**XYZ MONSTER IDENTIFICATION:**
• Black card frame
• Has Rank (left-aligned stars) instead of Level (right-aligned stars)
• Record as "Rank X" not "Level X"

**TOKEN IDENTIFICATION:**
• Gray card frame
• Not typically graded individually
• If detected, note as "Token" in card_type

**OLDER CARD VARIATIONS:**
• Pre-2004 cards may have different text formatting
• First print LOB cards are extremely valuable (especially 1st Edition)
• "Wavy" foil pattern on early Secret Rares differs from modern pattern
• SDK/SDY starter deck cards have no set rarity symbol

**CARD BACK VERIFICATION:**
• All authentic Yu-Gi-Oh! cards share the same brown swirl back design
• Any deviation in back design suggests proxy/counterfeit
• Japanese cards have the same back design as English cards
• The back should be consistent brown/gold spiral pattern
