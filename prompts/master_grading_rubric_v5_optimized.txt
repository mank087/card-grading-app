SECTION 3: HUMAN EYE GRADING STANDARD (v5.12)
---
**Core Principle:** Grade as a human expert would, using only visible evidence.

**PROHIBITED - Speculative Deductions:**
- "Deducting 0.5 for possible hidden defects under sleeve"
- "Cannot verify corner condition, assuming minor wear"
- "Glare may hide imperfections, scoring conservatively"
- "Conservatively scored due to holder obscuring view"

**CORRECT Handling of Visibility Limitations:**
1. Score based ONLY on what you CAN see
2. Adjust UNCERTAINTY (confidence_letter, grade_uncertainty), not the SCORE
3. Document the limitation honestly in notes
4. Lower confidence_letter if visibility is significantly impaired

**Example - WRONG:**
"Corner obscured by holder, deducting -0.5 conservatively"

**Example - RIGHT:**
"Corner partially obscured by holder. Visible portion shows sharp geometry with no fiber exposure detected. Confidence: B (reduced due to partial obstruction). Score: 9.5 based on visible area."

---
Perform THREE independent evaluations, then average for final grade.

**Pass Structure:**

**Defect Consensus Rule:**
- Defect seen in 2+ passes → confirmed defect, apply full deduction
- Defect seen in only 1 pass → re-examine carefully; if uncertain, apply smaller deduction
- High variance (>1.0) → increase grade_uncertainty

**Anti-Shortcut Rules:**
- Each pass must be genuinely independent analysis
- DO NOT simply copy Pass 1 scores to Passes 2 and 3
- Variance of exactly 0.0 across all passes for all components is suspicious
- Each pass should use slightly different examination approach

SECTION 14: SCORING CALCULATION
---
**14.1 Raw Sub-Scores (8 Components)**

All MUST be X.0 or X.5 increments only.
- Valid: 10.0, 9.5, 9.0, 8.5, 8.0, 7.5, etc.
- INVALID: 9.25, 8.63, 7.8, 9.3

**Rounding Rule:**
- x.00 to x.24 → round DOWN to x.0
- x.25 to x.74 → round to x.5
- x.75 to x.99 → round UP to (x+1).0

**14.2 Weighted Scoring**
- Front components × 0.55
- Back components × 0.45
- Combined category = front_weighted + back_weighted

**14.3 Weakest Link Calculation**

preliminary_grade = MIN(centering_weighted, corners_weighted, edges_weighted, surface_weighted)

**Floor Drag Rule:**
- If weakest ≥ 9.0: Use weakest (traditional)
- If weakest 8.0-8.5: MIN(weakest + 0.5, average - 0.5)
- If weakest < 8.0: MAX(weakest, average - 1.0)

**14.4 Defect Count Multiplier**

| Total Defects | Multiplier |
|---------------|------------|
| 1 | 1.0× |
| 2 | 1.1× |
| 3 | 1.15× |
| 4 | 1.2× |
| 5 | 1.25× |
| 6+ | 1.3× |

**14.5 Final Grade**
final_grade = MIN(rounded_preliminary_grade, any_grade_caps)

If alteration detected → final_grade = N/A
